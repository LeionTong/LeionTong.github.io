无效或已损坏的软件包 (PGP 签名)#

现象#

提示：无效或已损坏的软件包 (PGP 签名)。出现Keys错误，签名失败，导致 ArchLinux/Manjaro 无法正常升级或者安装软件。

方法1：

sud vim /etc/pacman.conf
...
[archlinuxcn]
SigLevel = Never   #SigLevel的值改为 Never 或 Optional TrustAll
...


sudo pacman -Syu

方法2： 1,移除旧的keys sudo rm -rf /etc/pacman.d/gnupg

2,初始化pacman的keys sudo pacman-key --init 3,加载签名的keys sudo pacman-key --populate archlinux archlinuxcn manjaro 4,刷新升级已经签名的keys sudo pacman-key --refresh-keys 5,清空并且下载新数据 sudo pacman -Sc 6,更新 sudo pacman -Syu
